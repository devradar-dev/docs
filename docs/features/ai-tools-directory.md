# AI Tools Directory

Explore and compare AI coding tools with detailed capability matrices. Find the perfect AI assistant for your development workflow.

## Overview

Access the directory from the **Tools** page in navigation. Each tool is comprehensively profiled with capabilities, platform support, privacy information, and pricing.

## Privacy Considerations

Understanding how AI tools handle your code is critical:

| Privacy Model | Description | Best For |
|---------------|-------------|----------|
| **Local-Only** | All processing on your machine | Sensitive codebases |
| **Cloud** | Code sent to external servers | Non-sensitive projects |
| **Hybrid** | Optional local processing | Flexible requirements |

**Tip:** Always verify privacy policies before using AI tools with proprietary code.
